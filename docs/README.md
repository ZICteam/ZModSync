## Скриншоты

Скриншоты в комплект пока не включены.

Причина:
- текущей текстовой документации достаточно для установки и поддержки
- для качественных скриншотов нужен отдельный воспроизводимый GUI-сеанс с заранее подготовленными состояниями интерфейса

Если понадобится, отдельно можно подготовить пакет скриншотов для:
- экрана списка серверов
- экрана прогресса загрузки
- экрана перезапуска
- примера ошибки входа без ModSync

## Правило обновлений

После каждого изменения в проекте нужно:
- поднять версию мода
- записать изменения в `CHANGELOG.md`
- обновить документацию, если изменение влияет на установку, использование или поддержку
