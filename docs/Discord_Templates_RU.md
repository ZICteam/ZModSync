# ModSync: шаблоны для Discord

## Короткое описание мода

```md
**ModSync** автоматически скачивает нужные моды, конфиги и клиентские файлы с сервера перед входом.  
Если чего-то не хватает или версия устарела, мод сам сравнит файлы, скачает нужное и попросит перезапуск, если это требуется.  
Для входа на сервер нужен **Forge 1.20.1 + ModSync**.
```

## Сообщение для канала установки

```md
## Установка ModSync

1. Установите **Minecraft Forge 1.20.1**
2. Положите файл **ModSync** в папку `mods`
3. Запустите игру
4. Зайдите в список серверов
5. Выберите сервер и нажмите **Скачать моды** или **Подключиться**

Если ModSync попросит перезапуск после загрузки файлов, перезапустите игру.
```

## Сообщение для техподдержки

```md
Если сервер не пускает:
1. Убедитесь, что установлен **Forge 1.20.1**
2. Убедитесь, что установлен **ModSync**
3. Нажмите **Обновить** в списке серверов
4. Если ModSync скачал файлы и попросил рестарт, перезапустите игру
5. Если ошибка осталась, отправьте:
   - скрин ошибки
   - скрин окна ModSync
   - файл `logs/modsync.log`
```

## Сообщение о требовании мода

```md
Для входа на сервер обязателен **ModSync**.

Если у вас нет ModSync или Forge, сервер может кикнуть вас с сообщением о том, что отсутствует обязательный мод синхронизации.
```

## Сообщение об ошибке хэша

```md
Ошибка `Invalid SHA-256` означает, что сервер отдал файл, который не совпадает с manifest.

Это обычно серверная проблема.
Если вы игрок, не переустанавливайте всё подряд, а отправьте лог `logs/modsync.log` в техподдержку.
```
