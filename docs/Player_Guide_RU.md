# ModSync: инструкция для игрока

## Что нужно для входа

- Minecraft `1.20.1`
- Forge `47.x`
- установленный мод `ModSync`

Без ModSync сервер может не пустить.

## Что делает ModSync

Перед входом на сервер ModSync:

- проверяет, хватает ли нужных файлов
- скачивает недостающие моды и клиентские файлы
- может попросить перезапустить игру

## Как зайти

1. Запусти игру с Forge
2. Убедись, что `ModSync` установлен
3. Открой список серверов
4. Выбери сервер
5. При необходимости нажми `Обновить`
6. Нажми `Скачать моды` или `Подключиться`

## Если ModSync что-то скачал

Если после загрузки появилось окно перезапуска:

- перезапусти Minecraft

Это нормально, особенно если были обновлены моды.

## Если не пускает

Проверь:

- установлен ли Forge `1.20.1`
- установлен ли ModSync
- есть ли место на диске
- не блокирует ли антивирус/фаервол загрузки

## Если появилась ошибка по хэшу

Если видишь в логах что-то вроде:

```text
Invalid SHA-256
Hash verification failed
```

это обычно проблема на стороне сервера.

Что отправить в поддержку:

- скрин окна ModSync
- скрин списка серверов
- файл `logs/modsync.log`

## Что отправить в техподдержку

```text
1. Версия Minecraft
2. Версия Forge
3. Есть ли ModSync в папке mods
4. Скрин ошибки
5. Лог logs/modsync.log
```
